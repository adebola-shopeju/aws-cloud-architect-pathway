Day 2 - Tuesday: Linux Filesystem + Permissions

Commands learned:
pwd, ls, ls -la, cd, mkdir, touch, cat, cp, mv, rm, nano

Key concepts:
- Linux filesystem is like a tree. Root is /
- Permissions = three traffic lights: Owner, Group, Everyone
- r=read, w=write, x=execute
- chmod 644 = I can edit, others can only read
- chmod 755 = I can do everything, others can run but not modify
